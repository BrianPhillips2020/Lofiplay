# LoFi Player
Opens default web-browser and open the [lofi-girl stream](https://www.youtube.com/watch?v=jfKfPfyJRdk). Built with gradle, requires [java 17](https://adoptium.net/)

## Build and Use
Simply build with
```
 ./gradlew build
```
and run `lofi.jar` from `build/libs`
